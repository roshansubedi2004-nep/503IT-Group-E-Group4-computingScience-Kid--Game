| Area                    | Original idea            | Improved idea                   | Reason for improvement   |
| ----------------------- | ------------------------ | ------------------------------- | ------------------------ |
| Instructions            | Simple quiz instructions | Clear step-by-step instructions | Easier to understand     |
| Visual style/background | Plain quiz screen        | Cyber safety background         | Looks more interesting   |
| Icons/buttons           | Few icons                | Lock, shield and key icons      | Helps users understand   |
| Feedback system         | Only right or wrong      | Short explanation after answer  | Helps users learn        |

| Reward system           | Simple score             | Points, stars and badges        | Makes users want to play |
